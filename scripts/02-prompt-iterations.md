## Iteration 1: Initial Prompt

**User Prompt:**
```
Create a 2-minute script for a lesson titled "What is a Prompt?"
Target audience: beginners
Include: definition, analogy (teacher giving instructions), a simple example, and a hook at the start
Use conversational, enthusiastic tone
```

**ChatGPT Response (First Draft):**
```
[HOOK]
Welcome to today's lesson! Have you ever wondered why some people get amazing results from AI while others get nothing useful? The secret is in how you ask!

[DEFINITION]
A prompt is simply the instructions you give to an AI tool. It's your question or command that tells the AI what you want it to do.

[ANALOGY]
Think of it like a teacher giving homework instructions. If a teacher says "write something" - students are confused! But if they say "write a 500-word essay about..." - everyone knows exactly what to do!

[EXAMPLE]
Weak: "Write about dogs"
Better: "Write a friendly guide to caring for golden retrievers for a first-time dog owner"
```

## 🧠 Key Learnings from Iterations

1. **Hook needs shock value** - First 15 seconds are crucial
2. **Repetition builds clarity** - Say it differently twice
3. **Examples beat theory** - Show, don't just tell
4. **Energy is contagious** - Enthusiastic script = enthusiastic delivery
5. **Transitions matter** - Always hint at what's next
