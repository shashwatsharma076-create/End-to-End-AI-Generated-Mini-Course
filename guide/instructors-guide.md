## 📋 Lesson Plan: 15 Minutes

### ⏱️ Minute 0-2: Hook (2 min)
**Activity:** Video - "The Problem with Bad Prompts"
- Show examples of bad prompts and useless AI responses
- Ask: "Has this happened to you?"
- **Bloom's:** None (engagement only)

### ⏱️ Minute 2-5: Definition (3 min)  
**Activity:** Video + Discussion
- Define: "Prompt engineering = giving clear instructions to AI"
- The Boss/Employee analogy
- **Bloom's:** Remember (define)

### ⏱️ Minute 5-10: Examples (5 min)
**Activity:** Video + Side-by-side comparisons
- School homework analogy
- Good vs Bad prompt examples
- **Bloom's:** Understand, Analyze

### ⏱️ Minute 10-12: Practice (2 min)
**Activity:** Hands-on exercise
- Learners try improving "Explain gravity"
- Share responses in chat
- **Bloom's:** Apply

### ⏱️ Minute 12-15: Quiz (3 min)
**Activity:** In-session quiz
- 5 questions
- Immediate feedback
- **Bloom's:** Evaluate

---

## 🧠 Bloom's Taxonomy Deep Dive

### Level 1: Remember (Define)
**Question:** What is prompt engineering?
**Evidence:** Quiz Q1 correct answer

### Level 2: Understand (Explain)  
**Question:** Why does it matter?
**Evidence:** Quiz Q3 explanation

### Level 3: Apply (Use)
**Question:** Write a better version of "Explain AI"
**Evidence:** Practice exercise completion

### Level 4: Analyze (Differentiate)
**Question:** Which is better and why?
**Evidence:** Quiz Q2 comparison

### Level 5: Evaluate (Judge)
**Question:** What's missing from this prompt?
**Evidence:** Quiz Q5 critique

### Level 6: Create (Design)
**Question:** Create prompts for your own use case
**Evidence:** Extension project

## 🔧 Adaptation Notes

### For Advanced Learners:
- Skip definition - dive straight to examples
- More complex practice prompts
- Extension: Chain-of-thought prompting

### For Struggling Learners:
- Repeat the boss analogy
- More examples before practice
- Pair work on prompts

### For Visual Learners:
- Use the slide comparison visuals
- Color-code good/bad prompts
- Diagram: prompt → AI → response
