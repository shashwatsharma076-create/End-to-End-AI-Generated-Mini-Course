### Video Settings

| Setting | Value |
|---------|-------|
| **Avatar** | "Instructor - Professional" |
| **Resolution** | 1920x1080 (HD) |
| **Frame Rate** | 30 fps |
| **Background** | Gradient (Purple → Dark) |
| **Lip Sync** | ON (auto-sync) |
| **Gestures** | ON (hand movements) |
| **Duration** | Match audio length |

### HeyGen Setup Steps

1. Go to **heygen.com**
2. Sign in → Studios
3. Select **"AI Avatar"**
4. Choose avatar (use "Instructor")
5. Upload audio (ElevenLabs MP3)
6. Enable: Lip Sync + Gestures
7. Generate video
8. Download MP4
