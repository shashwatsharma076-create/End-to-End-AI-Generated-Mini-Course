# 📚 End-to-End AI Generated Mini-Course

> A beginner-friendly course that teaches the fundamentals of AI prompting through interactive lessons.

## 📖 Course Structure

| Lesson | Topic | Duration |
|--------|-------|----------|
| 1 | What is a Prompt? | 2 minutes |
| 2 | Anatomy of a Good Prompt | 3 minutes |
| 3 | Practice: Writing Your First Prompt | 5 minutes |

---

## 🎯 Learning Objectives

By the end of this course, you will be able to:
- Define what a prompt is in the context of AI
- Understand why clear instructions matter
- Write effective prompts for AI tools

---

## 📝 How to Use This Course

1. Start with Lesson 1: "What is a Prompt?"
2. Watch the video/script
3. Complete the practice exercise
4. Move to next lesson
